The soft in software

We sometimes use the term hard coded when writing something like const withTaxes = sum * 1.19. The tax of 19% can change and if it does, we have to find all the spots where this number was used. In some cases the code might be written like this though: const total = sum * 19 / 100. So we would miss it if we searched the document for 1.19. In this context most developers would recognize what the number 19 means.
